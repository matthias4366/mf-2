Add a new nutrient profile

A nutrition profile is a collection of the dietary reference intakes for each nutrient - i.e. how much of it you should eat in a day. A good source for nutrient profiles for your gender is the National Institute of Health. It provides tables for the elements, the vitamins and macronutrients. If the links are broken, visit https://ods.od.nih.gov/Health_Information/Dietary_Reference_Intakes.aspx.

If you want to get done quickly, only add the macronutrients.

In the future, it will be possible to select nutrient profiles from a database.
